1989 SPRING

僕たち横浜万博警備隊

青い春とグレーの制服、A班の終わらない夜。

Author: 岩田 ヒロ

01. OVERVIEW

INFO.TXT

舞台設定： 1989年、昭和が終わり平成が始まった年の横浜。

横浜在住の大学生である主人公は、コンパニオンと友達になれるかもしれないという下心から「横浜万博」の警備員アルバイトに応募する。

線路脇や道路工事での事前研修を経て、高日給の「24時間ゲート警備（東口搬入ゲート）」に選抜される。そこで配属された「A班」の個性豊かで訳ありなメンバーたちと交流しながら、万博開幕直前の日々を過ごしていく青春群像劇。

STATUS: 連載中 (第1話公開済) | NEXT UPDATE: 2/27, 2/28, 3/1 20:00

02. 24H GATE SECURITY A-TEAM

ROOKIE

主人公（僕）

横浜在住の大学1年生。退屈やルーチンを嫌う。コンパニオン目当てでバイトを始めるが、育ってきた環境の違うA班の大人たちに囲まれ刺激的な日々を送る。

LEADER

結城さん

主人公と同い年のイケメン（180cm）。元暴走族リーダー。床屋の免許を持ち実家の手伝いをしている。冗談が面白く頼りがいのある性格。

INTEL

田村さん

関西出身で色黒の40歳くらい。本業は海外からお茶を輸入する卸問屋。面白そうだからと応募。暗算が速く英語も堪能なインテリ。

VETERAN

藤井さん

東北出身で細身の40歳くらい。元自衛隊員で吃音癖がある。バイト代を全て競馬とソープランドにつぎ込む破天荒だが、一番優しい性格。

DROPOUT

高橋

主人公の高校時代からの友人。一緒にバイトを始めたが、仕事がつまらないという理由で早々に辞めてしまう。
