# 200マイルの残像 - Webサイト構想設計書

## 1. プロジェクト概要
- **作品名**: 200マイルの残像
- **作者名**: 半熟たまご
- **ステータス**: 完結済（全17話）
- **ターゲット・テーマ**: 車、首都高、家族、90年代、上京、夜、バイク、走り屋

## 2. サイト構成案
シングルページ、もしくは2〜3ページのシンプルな構成。モダンでリッチなデザイン（ダークテーマ、ネオン、90年代の夜の高速道路をイメージしたグラスモーフィズムやリッチなアニメーション）を採用する。

### ページ一覧:
1. **Home (`index.html`)**
   - **Hero Section**: 夜の首都高を疾走する車の残像を表現したリッチなヘッダー画像/アニメーション。タイトルロゴ。
   - **Introduction**: あらすじ・概要（作品の魅力を伝えるセクション）。
   - **Episodes**: 全17話へのリンク集。3つの章立て（西岡編、長谷蔵編、終盤・結末）を視覚的に分離。
2. **Characters (`characters.html`)**
   - 主要キャラクターと搭乗マシンの紹介ページ。
   - スクロールに合わせて車やバイクのスペックがスライドインする動的表現。

- **Color Palette**:

## 4. 提供コンテンツ
### あらすじ
かつて東名高速で最速を誇った男「アニキ」が事故死してから十数年。彼の走りに魅了されていた西岡は、今も首都高で「アニキの残像」を追い求めていた。 一方、京都から上京してきた16歳の少年・黒崎健吾は、失踪した父と姉のルーツを探るため夜の首都高へ足を踏み入れる。時を同じくして、圧倒的な速さを誇る紫のR33 GT-Rに乗る真澄と、彼女の世界を理解するために走る若き外資系社長・長谷蔵も超高速の世界へ魅入られていく。 最高速の領域「200マイル（約320km/h）」で交錯する4台のマシン。彼らが限界を超えて踏み込んだ先で見つける「答え」とは――。首都高を舞台に、車と家族、それぞれの運命が絡み合うヒューマンドラマ。

- **黒崎 健吾（くろさき けんご）**: アニキの息子(16)。姉を探す。 / ホンダ CBR900rr（初代・赤青ツートン）

- **黒崎 真澄（くろさき ますみ）**: 健吾の姉(26)。狂気的な速さを求める。 / 日産 スカイライン R33 GT-R（ミッドナイトパープル・800馬力化された化け物マシン）
